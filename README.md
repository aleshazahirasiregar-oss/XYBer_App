# XYBer_App

Pusat repository untuk tweak dan optimasi sistem yang aman dan terproteksi.

## Tentang Komunitas
Komunitas ini dibentuk untuk mereka yang ingin mengoptimalkan perangkat Android dengan cara yang benar dan terukur. 

## Aturan GarudaXyber
1. Selalu lakukan backup sistem sebelum mencoba tweak apa pun.
2. Setiap modifikasi sistem harus melalui verifikasi/izin admin.
3. Keamanan perangkat adalah prioritas utama.
